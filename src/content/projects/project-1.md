## Project Overview

Bozsaat is a mobile application created to simplify access to real-time bus schedules for Bozüyük, Eskişehir, and Bilecik. The product goal was straightforward: help people check the next available bus quickly without navigating a cluttered or confusing interface.

## Objectives

1. Make local transit information easy to understand on mobile.
2. Reduce the time it takes to find useful schedule data.
3. Keep the product lightweight and simple enough for frequent daily use.

## Key Features

1. **Fast schedule lookup**
   - Users can quickly browse available bus times for supported routes and cities.

2. **Practical mobile-first UX**
   - The interface prioritizes readability and low-friction interactions so users can check schedules on the go.

3. **Data delivered through JSON sources**
   - Schedule information is loaded dynamically from web-hosted JSON data, making updates easier to manage.

## Technology Stack

- **Frontend:** Flutter for cross-platform UI and interaction design
- **Language:** Dart
- **Data source:** JSON delivered from the web

## Outcome

The app turned a repetitive local information problem into a cleaner mobile workflow. It gave users a faster way to plan travel and demonstrated how focused UX decisions can make utility products more valuable.
